# First

Утилита для извлечения четырёхзначных чисел из текста с интеллектуальной фильтрацией. Применяется для парсинга ставок оплаты из русскоязычных вакансий.

## Зависимости

- **NUnit** 3.14.0
- **NUnit3TestAdapter** 4.5.0
- **Microsoft.NET.Test.Sdk** 17.11.1
- Платформа: .NET 8.0

## Классы

### FourDigitNumberFinder

Основной класс. Метод `FindFourDigitNumbers(string input)` → `List<string>`.

**Алгоритм:**
1. Поиск всех четырёхзначных чисел по regex `\b\d{4}\b`
2. Фильтрация по правилам:

| Фильтр | Действие |
|--------|----------|
| ID вакансии | Исключить (маркеры: "id ", "id:", "№ ") |
| Год (1990-2030) | Исключить, если контекст указывает на год |
| Ставка в диапазоне годов | Включить, если контекст указывает на ставку |

### Контекстный анализ (IsRateContext)

Многоуровневая стратегия определения контекста числа:

**Уровень 1 - Маркеры года (перед числом, 30 символов):**
- "в ", "с ", "до ", "год", "году", "г. ", "year"

**Уровень 2 - Маркеры ставки после числа (200 символов):**
- Русские: "рублей", "руб/час", "руб.", "в час", "за час"
- Валюты: "$", "usd", "eur", "/hour", "/hr", "per hour"

**Уровень 3 - Маркеры ставки перед числом (100 символов):**
- "ставка", "оплата", "зарплата", "оклад", "rate", "hourly", "compensation", "от "

**Уровень 4 - Диапазоны ставок:**
- Паттерны: "1800/2270", "2500-3500"
- Разделители: `/`, `-`, `–`
- Проверка маркера ставки перед диапазоном

### HelloWorld

Точка входа. Метод `Main()` выводит "Hello world".

## Тесты (38 тестов)

| Категория | Тесты | Описание |
|-----------|-------|----------|
| Базовые | 1-8 | Простые числа, несколько чисел, с пунктуацией, пустые строки |
| Граничные | 9-14 | Ведущие нули, смешанный контент, пробелы, null |
| Годы vs Ставки | 15-19 | Различение 2000 как года и ставки |
| Многострочные | 20-30 | Маркер на следующей строке, табличный формат, английские маркеры |
| Диапазоны | 31-38 | Слэш, дефис, тире, полные примеры вакансий |

## Связь с основным проектом

Проект **First** - автономный, не является частью AnalyticsTC solution. Может использоваться как отдельная утилита для обработки текстов вакансий.
